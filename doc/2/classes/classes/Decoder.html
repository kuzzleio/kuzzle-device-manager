Class DecoderAbstract

Base class to implement a decoder for a device model. The device model must be passed to the parent constructor. The abstract "decode" method must be implemented.

measures

measures: NamedMeasures = []

Declaration of the measures decoded by this decoder. The name correspond of the measure name for the device. Measures types should be registered on the plugin beforehand.

Example

this.measures = [
  { type: 'temperature', name: 'temperatureExterior' },
];

Optional payloadsMappings

payloadsMappings?: JSONObject = {}

Custom mappings for the payload collection. It will be injected in the "payload" property and it should allows to index the device model unique identifier field.
